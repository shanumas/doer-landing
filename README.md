# doer-landing

Landing page for [doer](https://github.com/shanumas/doer) — autonomous GitHub issue fixer.

## Stack

Plain HTML + CSS + JavaScript. No framework needed — keep it fast and simple.

## Design brief

- Hero section: headline, subheadline, CTA button
- Features section: 3 key features (GitHub integration, Teams approval, autonomous PR)
- How it works: 3 steps (connect → approve → done)
- Footer with GitHub link
